# EYOUCMS 模板标签规划

> 本文档提供每种网站类型的页面标签速查，代码示例详见 `LLMS.md`。
>
> **使用流程**：用户提供栏目名称 → 匹配风格（[风格决策.md](风格决策.md)）→ 输出标签方案
>
> ⚠️ **常见问题**：[问题记录.md](问题记录.md)

## 一、内容发布网站

> **栏目结构**：3个文章栏目 + 单页关于我们
>
> **搜索**：直接套用文章列表页模板

### 1.1 公共片段

| 文件 | 核心标签 | 说明 |
| -- | -- | -- |
| `nav.htm` | `{eyou:models}`、`{eyou:searchform}` | 导航遍历、搜索表单 |
| `footer.htm` | `{eyou:global}`、`{eyou:links}`、`{eyou:navigation}` | 网站信息、友链、底部导航 |

### 1.2 页面模板

| 页面 | 核心标签 | 可选标签 | 特殊说明 |
| -- | -- | -- | -- |
| `index.htm` | `{eyou:artlist typeid='4/5/6'}` | `{eyou:adv}`、`{eyou:links}`、`{eyou:type}` | 3个栏目模块 + 1-3个额外模块 |
| `lists_article.htm` | `{eyou:list}`、`{eyou:pagelist}`、`{eyou:position}` | `{eyou:type}` | 无需指定 typeid，自动绑定当前栏目 |
| `view_article.htm` | `{$eyou.field.*}`、`{eyou:position}` | `{eyou:beafter}`、`{eyou:relevarticle}`、`{eyou:tags}`、`{eyou:arcclick}` | 使用全局字段变量 |
| `lists_single.htm` | `{$eyou.field.*}`、`{eyou:position}` | `{eyou:models}` | 单页直接用全局变量 |
| `lists_search.htm` | `{eyou:list}`、`{eyou:pagelist}` | — | `{$eyou.field.keywords}` 显示搜索词 |

## 二、商城网站

> **栏目结构**：3个产品栏目（ID: 1,2,3）+ 服务栏目（窄型文章列表）
>
> **搜索**：直接套用产品列表页模板

### 2.1 公共片段

| 文件 | 核心标签 | 说明 |
| -- | -- | -- |
| `nav.htm` | `{eyou:models}`、`{eyou:searchform modelid='2'}` | 导航遍历、商城产品搜索 |
| `footer.htm` | `{eyou:global}`、`{eyou:links}`、`{eyou:navigation}` | 网站信息、友链、底部导航 |

### 2.2 页面模板

| 页面 | 核心标签 | 可选标签 | 特殊说明 |
| -- | -- | -- | -- |
| `index.htm` | `{eyou:artlist typeid='1/2/3'}`、`{eyou:arcview}` | `{eyou:adv}`、`{eyou:links}` | 3个产品模块 + 1-3个额外模块 |
| `lists_product.htm` | `{eyou:list}`、`{eyou:pagelist}`、`{eyou:position}` | `{eyou:type}`、`{eyou:screening}` | 无需指定 typeid |
| `view_product.htm` | `{$eyou.field.*}`、`{eyou:sppurchase}` | `{eyou:attribute}`、`{eyou:beafter}` | 购买组件必选 |
| `lists_article_small.htm` | `{eyou:list}`、`{eyou:pagelist}`、`{eyou:position}` | `{eyou:type}` | 用于服务/条款等次要信息 |
| `view_article.htm` | `{$eyou.field.*}`、`{eyou:position}` | `{eyou:beafter}` | 服务详情页 |
| `lists_single.htm` | `{$eyou.field.*}`、`{eyou:position}` | — | 单页直接用全局变量 |
| `lists_search.htm` | `{eyou:list}`、`{eyou:pagelist}` | — | 商城搜索结果 |

## 三、首页额外模块

> 不依赖具体栏目，可用于任何类型网站，每次选用 1-3 个。

| 模块名称 | 推荐标签 | 用途 |
| -- | -- | -- |
| Banner 轮播 | `{eyou:adv pid='1' loop='5'}` | 广告位轮播图 |
| 服务优势 | `{eyou:artlist flag='c' loop='4'}` | 推荐文档展示 |
| 合作伙伴 | `{eyou:links type='image' loop='10'}` | 图片友链 |
| 新闻动态 | `{eyou:artlist typeid='文章栏目ID' loop='6' orderby='add_time'}` | 最新文章 |
| 产品展示 | `{eyou:artlist typeid='产品栏目ID' loop='8' orderby='click'}` | 热门产品 |
| 关于我们入口 | `{eyou:type typeid='单页ID'}` | 单页栏目链接 |
| TAG 标签云 | `{eyou:tags sort='rand' getall='1' loop='20'}` | 标签云展示 |
| 留言咨询入口 | `{eyou:guestbookform}` 或 `{eyou:form}` | 表单入口 |

## 四、标签使用总表

### 4.1 按使用频率排序

| 优先级 | 标签 | 用途 | 适用页面 |
| -- | -- | -- | -- |
| ★★★ | `{eyou:artlist}` | 文章/产品列表 | 首页、任何列表展示 |
| ★★★ | `{eyou:list}` | 列表页文档列表 | lists_* |
| ★★★ | `{eyou:models}` | 栏目导航 | nav、footer |
| ★★★ | `{eyou:position}` | 面包屑导航 | 所有内容页 |
| ★★★ | `{eyou:pagelist}` | 分页导航 | lists_* |
| ★★★ | `{$eyou.field.*}` | 字段变量 | view_* |
| ★★ | `{eyou:type}` | 栏目信息 | lists_*、index |
| ★★ | `{eyou:global}` | 全局配置 | 全站 |
| ★★ | `{eyou:searchform}` | 搜索表单 | nav |
| ★★ | `{eyou:adv}` | 广告轮播 | index |
| ★★ | `{eyou:links}` | 友情链接 | footer、index |
| ★★ | `{eyou:beafter}` | 上下篇 | view_* |
| ★★ | `{eyou:sppurchase}` | 商品购买 | view_product |
| ★ | `{eyou:attribute}` | 商品参数 | view_product |
| ★ | `{eyou:comment}` | 商品评价 | view_product |
| ★ | `{eyou:arcview}` | 单条文档 | 嵌套获取详情 |
| ★ | `{eyou:relevarticle}` | 相关文档 | view_* |
| ★ | `{eyou:arcclick}` | 点击量 | view_* |
| ★ | `{eyou:navigation}` | 导航菜单 | footer |
| ★ | `{eyou:tags}` | TAG 标签 | view_article |
| ★ | `{eyou:notempty}` / `{eyou:empty}` / `{eyou:if}` | 条件判断 | 通用 |

### 4.2 函数速查

| 函数 | 示例 | 用途 |
| -- | -- | -- |
| `MyDate` | `{$field.add_time\|MyDate='Y-m-d',###}` | 日期格式化 |
| `text_msubstr` | `{$field.title\|text_msubstr=###,0,30,true}` | 纯文本截取 |

## 五、标签嵌套注意事项

> ⚠️ **详细说明及实战示例**：[问题记录.md - 一、标签嵌套变量冲突](问题记录.md#一标签嵌套变量冲突)

**核心规则**：嵌套标签的内层必须定义 `id` 属性，避免 `{$field}` 变量冲突。

| 受影响标签 | 解决方案 |
| -- | -- |
| `{eyou:type}`、`{eyou:artlist}`、`{eyou:list}`、`{eyou:adv}`、`{eyou:links}`、`{eyou:tags}` | 内层添加 `id='xxx'`，变量改为 `{$xxx.*}` |

```html
{eyou:type typeid='1'}
  {$field.typename}
  {eyou:artlist typeid='1' id='item'}  <!-- 添加 id -->
    {$item.title}                        <!-- 使用 $item -->
  {/eyou:artlist}
{/eyou:type}
```
