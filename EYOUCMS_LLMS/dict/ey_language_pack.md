# 数据字典：ey_language_pack

> **表说明**：模板语言包变量表，存储多语言模板中使用的变量键值对
> **字段数量**：10
> **引用此表的标签**：[lang](../tags/lang.md)

---

## 字段列表

| 字段名 | 类型 | 可空 | 默认值 | 说明 |
|--------|------|:----:|--------|------|
| `id` 🔑 | int(10) | 否 | — | 主键，自增ID |
| `name` | varchar(50) | 否 | — | 变量名（模板中 `{eyou:lang name='xxx' /}` 的 xxx） |
| `value` | text | 否 | — | 变量值（对应语言的实际文本） |
| `is_syn` | tinyint(1) | 是 | 0 | 是否同步官方语言包：0=否，1=是 |
| `lang` | varchar(50) | 是 | cn | 所属语言标识 |
| `sort_order` | int(10) | 是 | 0 | 排序号 |
| `add_time` | int(11) | 是 | 0 | 新增时间（Unix时间戳） |
| `update_time` | int(11) | 是 | 0 | 更新时间（Unix时间戳） |

## 索引

| 键名 | 类型 | 唯一 | 字段 |
|------|------|:----:|------|
| PRIMARY | BTREE | 是 | id |
